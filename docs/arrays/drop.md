<h3 id="exportfunctiondroparrayarray-count"><code>export function drop *(array(array, [count])*</code></h3>

Remove N items from the beginning of the input array

#### Arguments
1. `array` *(T&#91;&#93;)*: input array
2. `[count]` *(number): number of items to drop &#42;(default `1`)*&#42;

#### Returns
*(&#42;)*: input array sans the dropped items

#### Example
```js
const result = arrays.drop([1, 2, 3], 2);
console.log(result);
> [3]
```
