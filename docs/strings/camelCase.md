<h3 id="camelcasestring"><code>camelCase([string=''])</code></h3>

camelCase updates a string to camelcase

#### Arguments
1. `[string='']` *(string)*: input string

#### Returns
*(string)*: returns new camelCase string

#### Example
```js
const result = strings.camelCase('--BEST_friend--');
console.log(result);
> 'bestFriend'
```
