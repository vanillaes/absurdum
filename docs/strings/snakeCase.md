<h3 id="snakecasestring"><code>snakeCase([string=''])</code></h3>

snakeCase updates a string to snakecase

#### Arguments
1. `[string='']` *(string)*: input string

#### Returns
*(string)*: returns new snakeCase string

#### Example
```js
const result = strings.snakeCase('--BEST--friend--');
console.log(result);
> 'best_friend'
```
